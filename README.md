# Unsupervised-Learning-Student-Analysis
Comprehensive unsupervised learning study on student performance. Includes Dimensionality Reduction (PCA, Factor Analysis) and Segmentation (Cluster &amp; Discriminant Analysis) using SAS --- Estudio integral de aprendizaje no supervisado sobre el rendimiento académico. Incluye Reducción de Dimensionalidad (ACP, Factorial) y Segmentación en SAS.
